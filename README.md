# Simple C++ Calculator

A simple console-based calculator implemented in C++.

## How to Use

1. Enter the first number.
2. Choose an operation (+, -, *, /).
3. Enter the second number.
4. View the result.

## Build and Run

Make sure you have CMake and a C++ compiler installed.

```bash
mkdir build
cd build
cmake ..
make
./SimpleCppCalculator
